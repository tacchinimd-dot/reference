================================================
  상품 레퍼런스 보드 - 사용 가이드
================================================

[ 처음 사용하는 경우 ]

1. Python 설치
   - https://www.python.org/downloads 접속
   - 최신 버전 다운로드 후 설치
   - 설치 화면에서 "Add Python to PATH" 체크 필수

2. 설치.bat 실행 (최초 1회)
   - 폴더 안의 "설치.bat" 파일을 더블클릭
   - 설치 완료 메시지가 나올 때까지 기다림
   - 인터넷 연결 필요 / 수 분 소요될 수 있음

------------------------------------------------

[ 매번 사용할 때 ]

1. 실행.bat 더블클릭
   - 검은 CMD 창이 열리고 서버가 시작됨
   - 잠시 후 브라우저가 자동으로 열림

2. 브라우저에서 상품 URL 입력
   - 상단 입력창에 상품 상세페이지 URL 붙여넣기
   - Enter 또는 "추가" 버튼 클릭
   - 이미지, 상품명, 가격, 소재가 카드로 표시됨

3. 카드 활용
   - 카드를 드래그해서 자유롭게 위치 이동
   - 카드 우상단 색상 점 클릭으로 카테고리 색상 지정
   - 카드 하단 메모란에 기획 메모 입력 가능
   - 상품명 클릭 시 원본 상품 페이지로 이동
   - 상품명·가격·소재 더블클릭으로 직접 수정 가능
   - 카드 내 "🔗 원본 보기" 클릭으로도 상품 페이지 이동
   - 이미지 클릭 시 직접 이미지 파일 업로드 가능

4. 기타 요소
   - T 버튼: 자유 텍스트 박스 추가
   - □ / ○ 버튼: 도형 추가 (크기·색상 조절 가능)
   - 탭 + 버튼: 새 탭 추가 / 탭 이름 더블클릭으로 변경
   - Ctrl+C / Ctrl+V: 선택한 요소 복사·붙여넣기
   - Delete / Backspace: 선택한 요소 삭제

5. 종료
   - CMD 창을 닫으면 서버도 자동 종료됨

------------------------------------------------

[ 팀원과 실시간 협업 ]

같은 와이파이(또는 사내 네트워크)에 있는 팀원과
보드를 실시간으로 공유할 수 있습니다.

1. 서버를 실행한 PC의 CMD 창에서 IP 주소 확인
     예) 같은 네트워크 팀원: http://192.168.0.10:5000

2. 팀원은 위 주소를 브라우저에 직접 입력
   (팀원 PC에 Python 설치 불필요)

3. 한 명이 카드를 추가·이동·수정하면 나머지
   화면에 즉시 반영됩니다.

- 좌측 상단 접속자 수 표시 (● 2명 접속 중)로
  현재 연결된 팀원 수를 실시간 확인 가능

------------------------------------------------

[ 참고 사항 ]

- 카드 정보는 서버에 자동 저장됩니다.
  - 로컬 실행 시: board_state.json 파일에 저장
  - Render 배포 시: PostgreSQL 데이터베이스에 저장 (영구 보존)
  실행.bat을 재시작해도 그대로 유지됩니다.

- 자동 파싱이 안 되는 경우 수동 입력 창이
  자동으로 열립니다. 상품 페이지에서 직접
  정보를 복사해 입력하면 됩니다.

- URL 없이도 "직접 입력" 버튼으로 빈 카드를
  수동 작성할 수 있습니다.

- 보드 좌측 상단의 서버 상태 표시
    ● 초록 (n명 접속 중) → 정상
    ● 빨강 (서버 꺼짐)   → 실행.bat 재시작 필요

- CMD 창은 서버 실행 중 항상 켜져 있어야 합니다.
  닫으면 파싱 기능과 협업 기능이 중단됩니다.

- 상품 추가 내역은 레퍼런스_백업.xlsx에 자동 기록됩니다.

------------------------------------------------

[ 파일 구성 ]

  설치.bat                      최초 1회 설치 실행 파일
  실행.bat                      매번 사용 시 실행 파일
  launch.py                     서버 실행 및 브라우저 오픈 스크립트
  product_board_server.py       서버 및 상품 파싱 스크립트
  product_reference_board.html  보드 화면
  board_state.json              보드 상태 자동 저장 파일 (자동 생성)
  레퍼런스_백업.xlsx             상품 추가 이력 백업 (자동 생성)
